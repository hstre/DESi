## Runtime change

`src/desi/logic/inference.py` adds one helper and one guard
inside `_try_causal_chain`:

```python
# Only the strong/categorical modals — 'may', 'might',
# 'could', 'would', 'must', 'should', 'shall', 'ought' are
# excluded because protected benchmarks use them as legal/
# medical hedging ('may constitute', 'must be evaluated').
_V47_MODAL_TOKENS = frozenset({
    "will", "cannot",
})
_V47_PAST_AUXILIARIES = frozenset({
    "was", "were", "had", "did",
})

def _is_past_observational_v47(text):
    # past auxiliary OR -ed suffix
    ...

def _modality_inconsistent(premises, conclusion):
    # conclusion has modal, no premise has the same modal,
    # AND all premises are past observational
    ...
```

Guard 19 is appended after the v4.5 guard. It:

* uses only *grammatical* closed-class tokens (modal
  auxiliaries and past auxiliaries) and one morphological
  cue (the ``-ed`` suffix),
* never inspects domain vocabulary — no marker tuple, no
  synonym list, no content words,
* fires iff the conclusion uses a modal verb that no premise
  introduces *and* every premise is past-observational.

No new InferenceRule, ClaimState, Frame, LedgerEvent, regex,
or marker bucket.

## Reflection

The directive asks: *"When does observation become
explanation — and when does it only imitate one?"*

Using only the v4.6 rescue data, the v4.7 reduction, the
contamination measurement, and the false-modality rate:

* **Ten of nineteen** v4.6 residue cases share the same
  grammatical signature: every premise is past-observational
  and the conclusion introduces a modal verb the premises
  never carried. Observation imitates explanation when a
  descriptive past tense is paired with a normative future
  in the conclusion — *without any premise warranting the
  shift*.
* **Zero of the protected pool** share that signature.
  Modal-shift is invariant across benchmark prose: valid
  causal chains either keep the conclusion in past/present
  observational tense or introduce the modal in a premise
  first.
* **Thirty-nine of forty** synthetic leaps fire the guard.
  **Zero of twenty** tense-consistent valid chains fire it.
* **Nine of nineteen** remaining cases (MISSING_BRIDGE_RULE)
  carry no modal shift — they are direct lexical
  contradictions between premise and conclusion. v4.7 does
  not address them, and v4.6 already showed no safe probe in
  its five-element bank could.

Observation becomes explanation when its temporal /
modal grammar matches across premises and conclusion.
Observation only imitates explanation when a past
descriptive premise is paired with a modal conclusion that
no premise itself carries. The discipline of v4.7 is to
insist on that grammatical match — using only closed
auxiliary sets, no domain vocabulary — and to keep the
false-modality rate at zero across the protected pool.
